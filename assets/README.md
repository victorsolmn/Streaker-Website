# Streaker Website Assets

This folder contains all the assets needed for the Streaker website.

## Required Assets:

### Images to Add:
1. **favicon.png** - 32x32px app icon
2. **app-dashboard.png** - 600x1200px screenshot of app dashboard
3. **screen-dashboard.png** - App dashboard screenshot
4. **screen-nutrition.png** - Nutrition screen screenshot
5. **screen-progress.png** - Progress screen screenshot
6. **screen-profile.png** - Profile screen screenshot
7. **user1.jpg**, **user2.jpg**, **user3.jpg** - User avatar images (60x60px)
8. **play-store.png** - Google Play Store badge (150x50px)
9. **app-store.png** - Apple App Store badge (150x50px)
10. **qr-code.png** - QR code for app download (150x150px)
11. **social-preview.png** - Open Graph preview image (1200x630px)
12. **demo-video.mp4** - Demo video of the app

## How to Generate Assets:

### App Screenshots:
1. Open the Streaker app on a phone/emulator
2. Take screenshots of each screen
3. Crop to phone screen dimensions (roughly 2:1 ratio)

### Store Badges:
Download official badges from:
- Google Play: https://play.google.com/intl/en_us/badges/
- App Store: https://developer.apple.com/app-store/marketing/guidelines/

### QR Code:
Generate using: https://www.qr-code-generator.com/
Link to: Your app's Play Store URL

### User Avatars:
Use placeholder images from:
- https://ui-avatars.com/
- https://pravatar.cc/

### Social Preview:
Create a 1200x630px image with:
- App name and logo
- Tagline
- Key features
- Phone mockup

## Placeholder Colors for Missing Images:
Until you add real images, the website will show gray placeholders.
The design will still work and look professional.
